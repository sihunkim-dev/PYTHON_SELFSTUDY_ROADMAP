# Console Calculator

## Main Objective
1. Parsing and Tokenizing User Input 
2. Application of Data Structure
3. Understanding of functional programming



## Workflow
1. User enter equation to solve
2. Parsing the user input 
   - set the parsing work flow with `list` in Python
   - for loop : check any operator in the list
3. Check the operator
   - set the switch statement to create a logic for each operator according to the priority

4. Display the result

- `q` for the exit

## Trouble shootings
- **Parsing** User Input
  -  Remove *white space*
  -  Set the *Priority*
  

## Execption Handling
- Check whether is numerical or not
  - `isdigit()` or `isspace()`
